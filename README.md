1. Clonar Repositorio
   git clone https://github.com/vanxgse/airbnb-app.git
3. Instalar dependencias
   npm install -y
5. Moverse a la carpeta client
   cd client/
7. Instalar dependencias de client
   npm install
9. Correr el proyecto
   Backend:
      npm run serverstart
   Frontend:
      npm start
